= CDATA - Replace =

Im [[lexer#pwLexer]] ist der erste Schritt die Entfernung aller CDATA-Sektionen.

== Syntax von CDATA ==

++''{{CDATA}}''++

Wobei ''Inhalt'' jede beliebiege Zeichenfolge sein kann.

== Darstellung von {{CDATA}} ohne "Match" ==

Zur Darstellung dieser Sequenz im Quellcode muss die Konstante ''<nowiki>{{CDATA}}</nowiki>'' verwendet werden.
