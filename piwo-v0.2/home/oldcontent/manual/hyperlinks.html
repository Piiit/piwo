Innerhalb dieser Datei

Vor dem Link ein # setzen. Alle Links können als Parameter einen neuen Anzeige-Namen erhalten.
! Identifizierung bzw. Ziel Befehl Ausgabe - Beispiel Hinweise
:: >> Kapitelauswahl
:ok: Kapitelnummer [[#1.2]] > 1.2 Falls die Kapitelnummer angegeben wird, wird der Name des Titels ausgegeben.
:ok: Kapitelname [[#Externe Links]] > externe links Alternativ kann auch der Titelname hergenommen werden.
:ok: Kapitelname (neu definiert) [[#Externe Links#Neuer Namen]] > externe links#neuer namen Der selbe Link mit einem neu definierten Linktext.
:ok: Fehlendes Kapitel [[#Nicht da]] > nicht da Fehlende Links werden markiert.
:: >> Spezielle Kürzel
(HINWEIS: Falls diese Kürzel als Kapitelnamen vorkommen, muss die Kapitelnummer als Sprungziel hergenommen werden.)
:ok: Anfang der Seite [[#TOP]] > top Springe zum Anfang der Seite (Defaulttext = Anfang).
:ok: Anfang der Seite (neu def.) [[#TOP#Hier ist der Anfang]] > top#hier ist der anfang Springe zum Anfang der Seite (Text wird als Parameter definiert).
:!: Inhaltsverzeichnis [[#PAGETOC]] > pagetoc Springe zum Inhaltsverzeichnis dieser Seite (Defaulttext = Inhalt).
:!: Hauptverzeichnis [[#TOC]] > toc Springe zum Inhaltsverzeichnis aller Seiten (Defaulttext = Inhalt).
:ok: Ende der Seite [[#BOTTOM]] > bottom Springe zum Ende der Seite (Defaulttext = Ende).
:?: Haupttitel [[#MAINTITLE]] > maintitle Springe zum Haupttitel (Defaulttext = Titel).

Innerhalb dieser Wiki

Wiki-Links haben einen Namespace, Kapitelnamen und optional einen Titel.
Zum Beispiel liegt dieser Wikitext im Namespace "reise" und heißt "fotoalbum". Daraus ergibt sich die vollständige ID: "reise:fotoalbum".
[[Namespace:Name#Text des Links]]
Falls nur der Name angegeben wird, wird das Wurzelverzeichnis hergenommen: 
[[Name#Text des links]]
:i: Außerdem gibt es noch spezielle Befehle im Namespace "pw_cms".
Dieser Namespace ist nur für berechtigungen#Administratoren zugänglich und enthält die Konfiguration für das CMS-Frontend.
