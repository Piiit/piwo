  # blablabla...
 # Eine Ebene tiefer
# ganz zurück

    1. Zweite Ebene
    2. Ein anderes Listenelement
  2. Noch eines auf der 1. Ebene
    1. Die zweite Liste auf der 2. Ebene (neu begonnen)
    2. Noch eines
      1. blablabla...
    3. Eine Ebene tiefer
  3. ganz zurück

Gemischte Liste

ACHTUNG: Listenelemente auf gleichem Level werden immer so geparst, wie die erste (=oberste) Angabe.
Siehe Beispiel...
# Das erste Element (Level 1: Definition, gültig für die ganze Liste)
 * hier wird das 2. Level definiert
# wieder zurück zum 1. Level
 * again
  * one
   * lala
 # zurück (wird als bullet geparst, obwohl ordered angegeben wurde)
# ganz zurück
  1. Das erste Element (Level 1: Definition, gültig für die ganze Liste)
    • hier wird das 2. Level definiert
  1. wieder zurück zum 1. Level
    • again
      • one
        • lala
    1. zurück (wird als bullet geparst, obwohl ordered angegeben wurde)
  2. ganz zurück
