Kernattribute

Attribute, die jedes Element hat.
  • required = true|false (default -> false)
  • html-element-id (default -> typ-name, z.B. float, int, bool, ... )
  • shortcuts (optional)
  • Label
  • Fehlermeldungen

integer Werte

  • von, bis
  • Einheit (z.B. ? oder Preis/kg)

float Werte

  • von, bis
  • Separator (point oder comma)
  • Anzahl der Dezimalstellen
  1. date Werte
  2. boolean Werte
  3. selects bzw. radiobuttons
  4. text und textfeld Werte
