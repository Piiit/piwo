    27| == Konstanten ==
    28| ; Hinweis
    29| : Konstanten können klein oder auch groß geschrieben werden.

    31| ; Schreibweise
    32| : ''%%{{Name der Konstante}}%%'' Einfache Notation ohne Unterkategorie oder Parameter.
    33| : ''%%{{Name der Konstante:Name der Unterkategorie}}%%'' Notation mit Unterkategorie (getrennt durch einen '':'')
    34| : ''%%{{Name der Konstante|Parameter1|Parameter2|...}}%%'' Parameter werden durch ''|'' voneinander getrennt.
